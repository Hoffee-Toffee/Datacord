Commands

Prefix Commands

Commands:

  • calc - WORK IN PROGRESS
  • game - WORK IN PROGRESS
  • getavatar - Returns a link to your avatar
  • getbot - Returns info on the bot
  • getcode - WORK IN PROGRESS
  • getemoji EMOJINAME- Displays foreign emoji
  • getemojilink EMOJI- Returns a link to the emoji
  • getlucky - Old easter egg
  • getreal VALUE - BotDev tool that displays raw text
  • help - Displays a link to this page
  • joke - Toggles joking
  • s QUERY - Searches in Google, Youtube and Wiki
  • savegame EMOJI- WORK IN PROGRESS

Terminology:

  • BOTNAME - The current name of the Bot
  • EMOJI - The name of the Emoji you want to use (e.g Thiccard, Bruh, Ow)
  • QUERY - The value you want to search (e.g What is Obama's last name?, Who is Joe?, the friendly lion yt)
  • VALUE - The value you want to use (e.g an emoji, a mention, a channel link)

Superhero commands

Commands:

  • getTABLE NAME - Displays a record from a table
  • listTABLE - Displays all record names from a table
  • listtables - Displays all table names from the database
  • newTABLE NAME - Creates an empty record (Except for the name provided) in a table
  • setTABLE NAME {FIELD=VALUE} - Changes a field in a table to a provided value

Terminology:

  • TABLE - The name of the Table you want to use (e.g supe, power, team)
  • NAME - The name of the Record you want to use (e.g The Traveler, Sedative Farts, The Avengers)
  • FIELD - The name of the Field you want to use (e.g Alias, Description, Members)
  • VALUE - The value you want to use (e.g The Traveler, Sedative Farts, The Avengers)

Build a bot

Setup

Application:

  • Go to the Discord Developer Portal
  • Make application
  • Make bot
  • Copy token
  • Exit

Glitch:

  • Go to the Glitch Homepage
  • Make an account
  • Make Discord Bot Project
  • Paste your token under DISCORD_TOKEN in .env
  • Click share, then live app, then the uppermost copy button

Uptime Robot:

  • Go to the Uptime Robot Homepage
  • Make an account
  • Make a monitor
  • Paste your link under URL
  • Copy main API key

Final Steps:

  • Paste API key into glitch
  • Add a prefix under .env
  • Go to dev portal
  • Create invite link
  • Test with command
